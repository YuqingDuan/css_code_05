CSS页面布局三大技术: 
	1.标准流技术normal flow 
	2.浮动技术float 
	3.定位技术position


各种页面布局技术总结:
	1.文档流(标准流)
		1.1 自上而下,自左向右
		1.2 块元素独占一行
		1.3 行内元素在一行排列,直到遇到父级元素边界换行

	2.浮动技术
		float: left | right
		2.1特点: 
			不占据原来的位置，会脱标。
			可以让块元素在一行上显示。
			可以将行内元素转成行内块。（不推荐使用）
		2.2清除浮动所产生的不利影响的三种方式
			2.2.1额外标签法
			<div style="clear:both;"></div>
			2.2.2 overflow:hidden;
			在父元素上使用overflow:hidden;
			2.2.3伪元素清除浮动(重点)
			/*固定式写法, 一次编写多处调用, 提高效率*/
			.clearfix:after{
				display: block;
				content: "."
				height: 0;
				line-height: 0;
				visibility: hidden;
				clear: both;
			}
			/*兼容IE浏览器*/
			.clearfix{
				zoom: 1;
			}
	3.定位技术
		position: left  | right |  top  |  bottom
		3.1静态定位
			position: static;  标准流
		3.2绝对定位(看脸型) position: absolute; 
			3.2.1绝对定位特点：
			◆自身从浏览器出发。
			◆绝对定位之后，元素不占位置，会脱标。
			◆嵌套的盒子，父元素没有给自身绝对定位，子元素绝对定位，子元素从浏览器出发设置自身位置。
			◆嵌套的盒子，父元素绝对定位，子元素绝对定位，子元素从父元素出发设置自身位置。
			◆行内元素转成行内块。（不推荐使用）
		3.3相对定位(自恋型) position: relative;
			3.3.1相对定位特点：
			◆位置从自身出发。
			◆设置相对定位之后，还占据原来的位置。
			◆子绝父相，子元素绝对定位，父元素相对定位。(重点)
			◆嵌套的盒子，父元素相对定位，子元素绝对定位，子元素从父元素出发设置自身位置。
			◆不能将行内元素转成行内块。
		3.4固定定位 position: fixed; 
			3.4.1固定定位特点：
			◆位置从浏览器出发。
			◆不占据原来的位置，会脱标
			◆可以将行内元素转换为行内块。（不推荐使用）
